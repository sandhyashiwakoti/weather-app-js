# Weather App

A responsive weather application that displays real-time weather information for any city worldwide.

## Live Demo
[weather-prototype3.vercel.app](https://weather-prototype3.vercel.app)

## Features
- Real-time weather data by city search
- Dynamic animated GIF backgrounds based on weather condition
- Glassmorphism UI design
- Temperature, humidity, pressure, wind speed & direction
- Offline mode with localStorage caching
- Online/offline toast notifications
- Loading and error states
- Responsive design

## Built With
- HTML5
- CSS3 (Glassmorphism, Flexbox, Media Queries)
- Vanilla JavaScript (Fetch API, Async/Await, localStorage)
- OpenWeatherMap API

## What I Learned
- Consuming third party REST APIs
- Async/Await and error handling
- localStorage for offline functionality
- Dynamic UI updates based on API response
- Glassmorphism CSS design

## Note
Originally built as a college project using PHP and MySQL backend.
Refactored to use OpenWeatherMap API directly from JavaScript.
